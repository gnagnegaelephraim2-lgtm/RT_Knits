Solution Design

NITA (Next-generation Intelligent Triage Assistant) replaces legacy FileMaker coordinator bottlenecks with an automated AI routing assistant for maintenance tasks at RT Knits factory in Mauritius.

Architecture

Stack: Rust (Actix-web) server + vanilla JS frontend + Supabase (PostgreSQL) + Vercel hosting. Integrates WhatsApp via n8n for Creole/English voice-to-task triage pipeline.
